# Calorie Counter

A simple web-based calorie counter application that allows users to select a restaurant and food item to view calorie information.

## Features

- Select a restaurant from a dropdown menu
- Dynamically load food items based on the selected restaurant
- Display calorie information for selected food items
- Clear selections and results easily
- User-friendly and responsive interface
- JSON-based food database

## Built With

- HTML
- CSS
- JavaScript
- JSON
